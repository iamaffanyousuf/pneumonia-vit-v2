# 🫁 Pneumonia Detection AI

A Vision Transformer-based model for detecting pneumonia from chest X-ray images.

## How to run

1. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```

2. Run the app:
   ```bash
   python app.py
   ```
